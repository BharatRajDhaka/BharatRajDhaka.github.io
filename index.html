Sound Control for Birds

A production‑ready, single html web app to control an ESP32 over MQTT via WebSockets (ws/wss). It publishes exact JSON commands (POWER, WINDOW, START/STOP, PLAY1/PLAY2, DELAY, VOLUME, STATUS) and streams live status/telemetry with a dark, responsive UI. Includes Mosquitto/EMQX presets, auth, MQTT 3.1.1/5, a timestamped console with mute/compact/auto‑scroll, and localStorage persistence—no build tools required.

Role: Full‑stack Developer • Tech: HTML, CSS, mqtt.js (CDN), MQTT over WebSockets (Mosquitto/EMQX), QoS 0/1, WS/WSS with reconnect and TLS guidance.
